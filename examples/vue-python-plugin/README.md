# Vue + Python 插件

## 🚀 功能特色

- **Vue 3 响应式前端**：使用最新的 Composition API
- **Python 数据处理后端**：强大的文本分析和处理能力
- **实时交互**：前后端通过 IPC 无缝通信
- **美观界面**：现代化的渐变设计和动画效果

## 🔧 功能模块

### 1. JSON 格式化
- 自动格式化 JSON 数据
- 错误检测和提示
- 统计信息显示

### 2. 数据验证
- 多种格式验证（JSON、邮箱、URL、数字）
- 中文字符检测
- 基础统计信息

### 3. 数据转换
- 大小写转换
- 字符串反转
- 去除空格和标点
- 提取数字和单词

### 4. 统计分析
- 字符和单词统计
- 字符类型分析
- 高频词汇统计
- 详细的数据报告

## 🛠️ 技术栈

- **前端**：Vue 3 + Composition API
- **后端**：Python 3 + 标准库
- **通信**：Electron IPC + 子进程
- **样式**：原生 CSS + 渐变动画

## 📦 安装使用

1. 在 UniHub 中打开插件管理
2. 拖拽 `plugin.zip` 到安装区域
3. 或使用 URL 安装：`http://localhost:8080/plugin.zip`

## 🎯 使用示例

```json
{
  "name": "测试数据",
  "value": 123,
  "items": ["a", "b", "c"]
}
```

选择不同的处理类型，体验 Vue + Python 的强大组合！
