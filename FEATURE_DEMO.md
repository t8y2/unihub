# 插件数据管理功能演示

## 功能亮点

### 🎯 核心功能

1. **智能备份** - 手动或自动备份插件数据
2. **一键恢复** - 从备份快速恢复插件状态
3. **灵活清理** - 清理数据或过期备份
4. **卸载策略** - 卸载时选择保留或删除数据

### 📊 数据统计

- 实时显示插件数据大小
- 统计备份数量和总大小
- 显示文件数量和最后修改时间

### 🔒 安全保障

- 所有删除操作需要确认
- 恢复前自动备份当前数据
- 完善的错误处理和提示

## 使用场景

### 场景 1: 插件升级

```
1. 备份当前插件数据
2. 卸载旧版本插件
3. 安装新版本插件
4. 如果新版本有问题，从备份恢复
```

### 场景 2: 临时卸载

```
1. 卸载插件时勾选"保留数据"
2. 系统自动创建备份
3. 需要时重新安装插件
4. 从备份恢复数据
```

### 场景 3: 数据迁移

```
1. 在旧设备上备份插件数据
2. 导出备份文件
3. 在新设备上安装插件
4. 导入并恢复备份
```

### 场景 4: 定期维护

```
1. 定期清理不需要的插件数据
2. 清理 30 天前的过期备份
3. 释放磁盘空间
```

## 技术实现

### 架构设计

```
┌─────────────────────────────────────────┐
│           Renderer Process               │
│  ┌─────────────────────────────────┐    │
│  │  PluginDataManager.vue          │    │
│  │  - 数据展示                      │    │
│  │  - 用户交互                      │    │
│  └─────────────────────────────────┘    │
│              ↓ IPC                       │
├─────────────────────────────────────────┤
│            Main Process                  │
│  ┌─────────────────────────────────┐    │
│  │  plugin-data-manager.ts         │    │
│  │  - 备份/恢复逻辑                 │    │
│  │  - 文件操作                      │    │
│  │  - 元数据管理                    │    │
│  └─────────────────────────────────┘    │
│              ↓                           │
│  ┌─────────────────────────────────┐    │
│  │  File System                     │    │
│  │  - plugins/                      │    │
│  │  - plugin-backups/               │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

### 数据流

```
备份流程:
用户点击备份 → IPC 调用 → 读取插件目录 →
创建 ZIP → 保存备份文件 → 更新元数据 →
返回结果 → UI 更新

恢复流程:
用户点击恢复 → IPC 调用 → 备份当前数据 →
解压备份文件 → 覆盖插件目录 → 返回结果 →
UI 更新 → 触发插件重载
```

## 代码示例

### 备份插件数据

```typescript
// 渲染进程
const backupPlugin = async (pluginId: string) => {
  try {
    const result = await window.api.pluginData.backup(pluginId)
    if (result.success) {
      toast.success(result.message)
      console.log('备份 ID:', result.backupId)
    }
  } catch (error) {
    toast.error('备份失败')
  }
}
```

### 恢复备份

```typescript
// 渲染进程
const restoreBackup = async (backupId: string) => {
  try {
    const result = await window.api.pluginData.restore(backupId)
    if (result.success) {
      toast.success(result.message)
      // 触发插件重新加载
      window.dispatchEvent(new CustomEvent('plugin-data-restored'))
    }
  } catch (error) {
    toast.error('恢复失败')
  }
}
```

### 卸载插件（保留数据）

```typescript
// 渲染进程
const uninstallPlugin = async (pluginId: string) => {
  try {
    const result = await window.api.plugin.uninstall(pluginId, {
      keepData: true // 保留数据
    })

    if (result.success) {
      toast.success(result.message)
      if (result.backupId) {
        console.log('数据已备份:', result.backupId)
      }
    }
  } catch (error) {
    toast.error('卸载失败')
  }
}
```
