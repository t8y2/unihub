# 内置工具

UniHub 内置了多种常用的开发工具，无需安装即可使用。

## 编码/解码工具

### Base64 编解码

将文本或文件转换为 Base64 编码，或将 Base64 解码为原始内容。

**功能特点**:

- 支持文本和文件编码
- 支持图片 Base64 预览
- 一键复制结果

### URL 编解码

对 URL 进行编码或解码处理。

**功能特点**:

- 支持完整 URL 编码
- 支持组件编码（encodeURIComponent）
- 自动检测编码类型

### Unicode 转换

在 Unicode 和中文之间进行转换。

**功能特点**:

- 支持 `\uXXXX` 格式
- 支持 `&#xXXXX;` HTML 实体格式

## 格式化工具

### JSON 格式化

美化、压缩和校验 JSON 数据。

**功能特点**:

- 语法高亮显示
- 自动格式化
- JSON 校验和错误提示
- 支持压缩输出

### JavaScript 格式化

格式化和压缩 JavaScript 代码。

**功能特点**:

- 基于 Prettier 格式化
- 支持代码压缩
- 可配置格式化选项

## 加密/哈希工具

### 哈希计算

计算文本或文件的哈希值。

**支持算法**:

- MD5
- SHA1
- SHA256
- SHA512

### JWT 工具

编码、解码和验证 JWT Token。

**功能特点**:

- 解析 JWT 结构
- 验证签名
- 支持 HS256/HS384/HS512 算法

## 时间工具

### 时间戳转换

在 Unix 时间戳和日期时间之间转换。

**功能特点**:

- 支持秒/毫秒时间戳
- 多种日期格式
- 显示当前时间戳

## 文本工具

### 正则测试

在线测试正则表达式。

**功能特点**:

- 实时匹配高亮
- 支持全局/多行/忽略大小写标志
- 显示匹配组

### 文本对比

对比两段文本的差异。

**功能特点**:

- 逐行对比
- 高亮显示差异
- 支持忽略空白

## 生成工具

### UUID 生成

生成各版本的 UUID。

**支持版本**:

- UUID v1（基于时间）
- UUID v4（随机）
- UUID v7（基于时间，可排序）

### 随机字符串

生成随机字符串和密码。

**功能特点**:

- 可配置长度
- 可选字符集（大小写字母、数字、特殊字符）
- 批量生成

## 颜色工具

### 颜色转换

在不同颜色格式之间转换。

**支持格式**:

- HEX (#RRGGBB)
- RGB (r, g, b)
- HSL (h, s%, l%)
- HSV (h, s%, v%)

## 其他工具

### 二维码生成

生成和解析二维码。

**功能特点**:

- 文本/URL 生成二维码
- 支持自定义尺寸和颜色
- 下载二维码图片

### 图片压缩

压缩图片文件大小。

**功能特点**:

- 支持 JPG/PNG/WebP
- 可调节压缩质量
- 批量压缩
