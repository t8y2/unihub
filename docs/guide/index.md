# 简介

UniHub 是一个面向开发者的工具集合平台，旨在将常用的开发工具整合到一个应用中，提升开发效率。

## 为什么选择 UniHub？

### 🎯 一站式工具箱

不再需要在多个网站或应用之间切换，UniHub 将 JSON 格式化、Base64 编解码、时间戳转换等常用工具整合在一起。

### 🔌 可扩展的插件系统

UniHub 提供了强大的插件系统，你可以：

- 从插件商店安装第三方插件
- 开发自己的插件满足特定需求
- 分享插件给其他开发者

### 🔒 本地优先，隐私安全

- 所有工具本地运行，数据不上传
- 插件权限隔离，保护系统安全
- 开源透明，代码可审计

### ⚡ 原生性能

基于 Electron + Vue 3 构建，享受原生应用的性能和体验。

## 技术栈

- **框架**: Electron 36 + Vue 3
- **构建工具**: Vite + electron-vite
- **UI**: Tailwind CSS + shadcn/vue
- **语言**: TypeScript
- **包管理**: pnpm

## 系统要求

- **macOS**: 10.15 (Catalina) 或更高版本
- **Windows**: Windows 10 或更高版本
- **Linux**: Ubuntu 18.04 或同等版本
