# 插件管理

UniHub 支持通过插件扩展功能。你可以从插件商店安装插件，也可以安装本地插件。

## 插件商店

### 浏览插件

1. 点击侧边栏的 **插件管理**
2. 切换到 **插件商店** 标签
3. 浏览可用的插件列表

### 安装插件

1. 在插件商店中找到想要的插件
2. 点击 **安装** 按钮
3. 等待下载和安装完成
4. 插件会自动出现在侧边栏中

### 更新插件

当插件有新版本时，插件管理页面会显示更新提示。点击 **更新** 按钮即可更新到最新版本。

## 已安装插件

### 查看已安装插件

1. 点击侧边栏的 **插件管理**
2. 在 **已安装** 标签中查看所有已安装的插件

### 启用/禁用插件

点击插件旁边的开关可以启用或禁用插件。禁用的插件不会出现在侧边栏中。

### 卸载插件

1. 在已安装列表中找到要卸载的插件
2. 点击 **卸载** 按钮
3. 确认卸载

## 官方插件

以下是一些推荐的官方插件：

### Markdown 编辑器

基于 Vditor 的 Markdown 编辑器，支持：

- 所见即所得编辑
- 实时预览
- 数学公式（LaTeX）
- Mermaid 图表
- 导出 HTML/PDF

### 代码对比工具

基于 Monaco Editor 的代码差异对比工具：

- 语法高亮
- 并排对比
- 内联对比
- 支持多种编程语言

### 剪贴板历史

记录和管理剪贴板历史：

- 自动记录复制内容
- 搜索历史记录
- 一键重新复制
- 支持文本和图片

### Excalidraw 绘图

手绘风格的图表绘制工具：

- 自由绘制
- 形状和箭头
- 文本标注
- 导出图片

### Ctool 工具集

包含 40+ 开发工具的工具集：

- 哈希/加密
- 编码转换
- 代码格式化
- 网络工具
- 更多...

## 插件权限

插件可能需要以下权限：

| 权限        | 说明         |
| ----------- | ------------ |
| `clipboard` | 读写剪贴板   |
| `storage`   | 本地数据存储 |
| `fs`        | 文件系统访问 |
| `http`      | 网络请求     |
| `spawn`     | 执行系统命令 |

安装插件时，请注意查看插件所需的权限，只安装来自可信来源的插件。
