# 快速上手

本指南将帮助你快速了解 UniHub 的基本使用方法。

## 界面概览

UniHub 的界面分为以下几个部分：

```
┌─────────────────────────────────────────────────┐
│  标题栏 / 标签栏                                  │
├──────────┬──────────────────────────────────────┤
│          │                                      │
│  侧边栏   │           内容区域                    │
│          │                                      │
│  - 主页   │                                      │
│  - 收藏   │                                      │
│  - 最近   │                                      │
│  - 工具   │                                      │
│          │                                      │
├──────────┴──────────────────────────────────────┤
│  底部工具栏（主题切换、设置）                       │
└─────────────────────────────────────────────────┘
```

## 打开工具

有多种方式打开工具：

### 1. 从侧边栏选择

点击侧边栏中的工具名称，即可在新标签页中打开该工具。

### 2. 使用全局搜索

按下 `⌘K`（macOS）或 `Ctrl+K`（Windows/Linux）打开全局搜索，输入工具名称快速定位。

### 3. 从主页选择

主页展示了最近使用和收藏的工具，点击即可快速访问。

## 标签页管理

UniHub 使用标签页来管理多个工具：

- **新建标签**: `⌘N` 或点击标签栏的 `+` 按钮
- **关闭标签**: `⌘W` 或点击标签上的关闭按钮
- **切换标签**: 点击标签或使用快捷键
- **右键菜单**: 右键点击标签可以关闭其他标签、关闭左侧/右侧标签

## 收藏工具

对于常用的工具，你可以将其添加到收藏：

1. 在主页或工具列表中，点击工具旁边的 ❤️ 图标
2. 收藏的工具会显示在侧边栏的"收藏"分类中

## 主题切换

UniHub 支持浅色和深色主题：

- 点击侧边栏底部的 🌙/☀️ 图标切换主题
- 或在 **设置 → 外观** 中选择主题

## 快捷键

常用快捷键：

| 快捷键 | 功能         |
| ------ | ------------ |
| `⌘K`   | 打开全局搜索 |
| `⌘N`   | 新建主页标签 |
| `⌘W`   | 关闭当前标签 |
| `⌘B`   | 切换侧边栏   |
| `⌘,`   | 打开设置     |
