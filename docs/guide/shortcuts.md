# 快捷键

UniHub 提供了丰富的快捷键支持，帮助你更高效地使用应用。

## 全局快捷键

这些快捷键在应用的任何位置都可以使用：

| 快捷键          | 功能                  |
| --------------- | --------------------- |
| `⌘ Shift Space` | 显示/隐藏 UniHub 窗口 |
| `⌘K`            | 打开全局搜索          |

::: tip
全局快捷键即使在 UniHub 不是前台应用时也可以使用。
:::

## 应用快捷键

| 快捷键   | 功能                |
| -------- | ------------------- |
| `⌘N`     | 新建主页标签        |
| `⌘W`     | 关闭当前标签        |
| `⌘B`     | 切换侧边栏显示/隐藏 |
| `⌘,`     | 打开设置            |
| `Escape` | 关闭搜索/弹窗       |

## 标签页快捷键

| 快捷键 | 功能         |
| ------ | ------------ |
| `⌘W`   | 关闭当前标签 |
| `⌘N`   | 新建标签     |

## 自定义快捷键

你可以在设置中自定义部分快捷键：

1. 打开 **设置**（`⌘,` 或点击侧边栏底部的齿轮图标）
2. 选择 **快捷键** 选项卡
3. 点击要修改的快捷键
4. 按下新的快捷键组合
5. 点击保存

### 可自定义的快捷键

- 显示/隐藏窗口
- 全局搜索

::: warning 注意
某些快捷键可能与系统或其他应用冲突。如果快捷键不生效，请尝试更换其他组合。
:::

## macOS 与 Windows/Linux 对照

| macOS | Windows/Linux | 功能            |
| ----- | ------------- | --------------- |
| `⌘`   | `Ctrl`        | Command/Control |
| `⌥`   | `Alt`         | Option/Alt      |
| `⇧`   | `Shift`       | Shift           |
| `⌘K`  | `Ctrl+K`      | 全局搜索        |
| `⌘N`  | `Ctrl+N`      | 新建标签        |
| `⌘W`  | `Ctrl+W`      | 关闭标签        |
| `⌘B`  | `Ctrl+B`      | 切换侧边栏      |
| `⌘,`  | `Ctrl+,`      | 打开设置        |
