# 主题设置

UniHub 支持浅色和深色两种主题，你可以根据喜好或环境光线选择合适的主题。

## 切换主题

### 方式一：侧边栏按钮

点击侧边栏底部的主题切换按钮：

- ☀️ 太阳图标：当前为深色主题，点击切换到浅色
- 🌙 月亮图标：当前为浅色主题，点击切换到深色

### 方式二：设置页面

1. 打开 **设置**
2. 选择 **外观** 选项卡
3. 在主题选项中选择：
   - 浅色
   - 深色

## 主题效果

### 浅色主题

浅色主题适合在光线充足的环境下使用：

- 白色背景
- 深色文字
- 柔和的阴影效果

### 深色主题

深色主题适合在光线较暗的环境下使用，可以减少眼睛疲劳：

- 深灰色背景
- 浅色文字
- 降低屏幕亮度对眼睛的刺激

## 插件主题适配

UniHub 会自动将当前主题通知给所有插件。插件开发者可以监听主题变化事件，实现主题适配。

```javascript
// 插件中监听主题变化
window.addEventListener('theme-changed', (event) => {
  const theme = event.detail // 'light' 或 'dark'
  // 更新插件 UI
})
```

## 主题持久化

你选择的主题会自动保存，下次启动 UniHub 时会恢复上次的主题设置。
