# 网站导航组件使用指南

## 功能介绍

网站导航是一个内置工具，帮助你收藏和管理常用网站，类似于浏览器书签，但更加简洁和易用。

## 主要功能

### 1. 添加网站

- 点击右上角的"添加网站"按钮
- 填写网站信息：
  - **网站名称**（必填）：例如 "GitHub"
  - **网站地址**（必填）：例如 "github.com" 或 "https://github.com"
  - **图标地址**（可选）：支持 `https://` 或 `data:image/` 格式的图片链接
  - **分类**（可选）：例如 "开发工具"、"社交媒体" 等

> **提示**：如果不填写图标地址，系统会自动显示网站名称的首字母，并使用彩色渐变背景。

### 2. 打开网站

- 直接点击网站卡片即可在默认浏览器中打开

### 3. 编辑网站

- 鼠标悬停在网站卡片上
- 点击右上角的编辑图标（铅笔）
- 修改信息后保存

### 4. 删除网站

- 鼠标悬停在网站卡片上
- 点击右上角的删除图标（垃圾桶）
- 确认删除

### 5. 搜索网站

- 使用顶部搜索框输入关键词
- 支持搜索网站名称、地址和分类

### 6. 分类筛选

- 添加网站时可以设置分类
- 顶部会自动显示所有分类标签
- 点击分类标签可以快速筛选

## 使用技巧

1. **自动获取图标**：
   - 添加网站后会自动从 Google Favicon API 获取图标
   - 图标会转换为 data URL 并缓存在本地
   - 如果自动获取失败，会显示网站名称首字母的彩色图标
2. **URL 格式**：输入网址时可以省略 `https://`，系统会自动添加
3. **分类管理**：建议使用统一的分类名称，如"开发工具"、"设计资源"、"学习平台"等
4. **快速访问**：将常用网站添加到导航，可以快速访问而无需打开浏览器书签
5. **自定义图标**：可以手动指定图标 URL，支持 https:// 或 data:image/ 格式

## 数据存储

- 所有网站数据保存在浏览器的 localStorage 中
- 数据会自动持久化，关闭应用后不会丢失
- 存储键：`web-navigator-sites`

## 示例分类

建议的分类示例：

- 开发工具
- 设计资源
- 学习平台
- 社交媒体
- 新闻资讯
- 在线工具
- 娱乐休闲
- 工作协作

## 技术实现

- 使用 `localStorage` 进行数据持久化（内置插件）
- 使用 `window.unihub.system.openExternal` 打开外部链接
- 使用 `window.unihub.http` API 获取网站图标（绕过 CSP 限制）
- 自动将图标转换为 base64 data URL 并缓存
- 使用 Google Favicon API 获取网站图标
- 响应式网格布局，支持多种屏幕尺寸
