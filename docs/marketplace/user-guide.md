# 插件市场使用指南

## 🏪 什么是插件市场

UniHub 插件市场是一个集中的插件分发平台，让你可以轻松发现、安装和管理插件。

## 🔍 浏览插件

### 在应用内浏览

1. 打开 UniHub
2. 点击侧边栏的「插件管理」
3. 切换到「插件商店」标签
4. 浏览可用插件

### 搜索插件

- **关键词搜索**: 在搜索框输入关键词
- **分类筛选**: 按工具、格式化、编码等分类筛选
- **作者筛选**: 查找特定作者的插件

## 📦 安装插件

### 从市场安装

1. 在插件市场中找到想要的插件
2. 点击「安装」按钮
3. 等待下载和安装完成
4. 插件会自动出现在侧边栏

### 手动安装

1. 下载插件的 `.zip` 文件
2. 进入「插件管理」→「本地插件」
3. 点击「安装本地插件」
4. 选择下载的 `.zip` 文件

## 🔧 管理插件

### 启用/禁用插件

1. 进入「插件管理」→「已安装」
2. 找到目标插件
3. 使用开关控制启用状态

### 更新插件

1. 当有新版本时，插件会显示「更新」按钮
2. 点击「更新」自动下载新版本
3. 更新完成后会自动重启插件

### 卸载插件

1. 进入「插件管理」→「已安装」
2. 找到要卸载的插件
3. 点击「卸载」按钮
4. 确认卸载操作

## 🛡️ 安全提示

### 权限确认

安装插件时，系统会显示插件需要的权限：

- **剪贴板**: 读写剪贴板内容
- **数据库**: 本地数据存储
- **文件系统**: 访问文件和目录
- **网络**: 发送 HTTP 请求
- **系统**: 系统通知、信息获取

### 安全建议

- ✅ 只安装来自可信来源的插件
- ✅ 仔细检查插件权限
- ✅ 定期更新插件到最新版本
- ❌ 不要安装来源不明的插件

## ❓ 常见问题

### Q: 插件安装失败怎么办？

A: 检查网络连接，确保下载链接有效。如果问题持续，尝试手动下载安装。

### Q: 插件占用太多内存？

A: 可以在插件管理中禁用不常用的插件，或者联系插件作者优化。

### Q: 如何报告插件问题？

A: 在插件详情页面找到「反馈」链接，或者到插件的 GitHub 仓库提交 Issue。

### Q: 插件数据会丢失吗？

A: 插件数据存储在本地，卸载插件时可以选择是否保留数据。
