# 图片压缩工具

一个简单高效的图片压缩插件，支持批量处理、格式转换和尺寸调整。

## 功能特性

- 📦 **批量压缩** - 一次处理多张图片
- 🎨 **格式转换** - 支持 JPG、PNG、WebP 互转
- 📏 **尺寸调整** - 自定义最大宽度
- 🎯 **质量控制** - 10%-100% 可调节
- ⚡ **纯前端处理** - 无需上传，保护隐私
- 📊 **实时预览** - 查看压缩前后对比

## 使用方法

1. 拖拽图片到上传区域或点击上传
2. 调整压缩参数（质量、尺寸、格式）
3. 点击"压缩并下载全部"
4. 压缩后的图片会自动下载

## 技术栈

- Vue 3
- TypeScript
- Tailwind CSS
- browser-image-compression

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 打包插件
pnpm package
```
